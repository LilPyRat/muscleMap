# muscleMap
Программа для помощи клиентам в проффесиональной тренировки в зале.
Запоминает дату тренироввки, карту мышц, ай ди пользователя.

## Функции

1. **Добавление упражнения:**
   - Выберите упражнение из списка
   - Введите вес (кг)
   - Введите количество повторений
   - Нажмите "Добавить"

2. **Расчет нагрузки:**
   - Добавьте одно или несколько упражнений
   - Нажмите "Рассчитать нагрузку"
   - Увидите интенсивность нагрузки на каждую мышечную группу

3. **Сохранение тренировки:**
   - Нажмите "Сохранить тренировку"
   - Данные сохранятся в базу данных

4. **Просмотр истории:**
   - Нажмите "Загрузить историю" в нижней карточке

## Описание функционала

### Backend (Python + Flask)
- REST API endpoints
- Работа с PostgreSQL через psycopg2
- Автоматическое создание таблиц и заполнение базовыми упражнениями
- Расчет интенсивности нагрузки

### Frontend (JavaScript)
- Интерактивный интерфейс
- Динамическая загрузка упражнений
- Добавление/удаление упражнений
- Отображение результатов расчета
- История тренировок

### База данных (PostgreSQL)
- Таблица `exercises` - справочник упражнений
- Таблица `workout_entries` - записи о тренировках

## Формула расчета интенсивности

```
Интенсивность = (вес × повторения × коэффициент_упражнения) / 100
```

Где коэффициент зависит от сложности упражнения:
- Простые изолирующие: 0.5-0.7
- Средние: 0.8-1.0
- Сложные базовые: 1.1-1.3

Уровни интенсивности:
- Низкая: < 30
- Средняя: 30-60
- Высокая: 60-90
- Очень высокая: > 90
